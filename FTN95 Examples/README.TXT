 The program in this directory is provided so that you can verify correct
 installation of Salford FTN95. 

 DEMO95.F90

 An FTN95 demonstration program which calculates some simple statistics.
 This program is written in free format.

 INSTRUCTIONS:

 o Type MK95 to make executables for the example with FTN95. If this BAT
   file works, you have verified installation of FTN95.

 o Run the DOS Program DEMO95.EXE. The program will produce several lines
   of text results.
